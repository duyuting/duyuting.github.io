ps

map与unordered_map的区别和联系
STL中map对应的数据结构是红黑树，红黑树的数据是有序的，查找时间复杂度是O(log(N)),而unordered_map的对应hash表，哈希表的特点是查找效率高，因此遇到查找问题就考虑下unordered_map，时间复杂度是O(1),额外的空间复杂度要高出许多。关于进一步说明可以参见博客。
